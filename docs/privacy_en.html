Privacy Policy

1. Information Collection

Atujii is a local image management tool, and we commit to:

  • All images and video files are stored only on your local device
  • No personal data is collected or uploaded to any servers
  • AI tagging functionality runs entirely locally, with no images sent to the cloud

2. Data Storage

Your data security is our top priority:

  • All media files remain in your local storage
  • Application settings and configurations are saved locally only
  • No user data is stored in the cloud

3. Permission Usage

Atujii requires the following permissions to provide full functionality:

  • File system access: For reading and managing your media files
  • Network permission: Only used for software update checks
  • System notification permission: For displaying operation completion notifications

4. Updates

We may periodically update this privacy policy. Updated policies will be published on this page with the last update date. We recommend checking this policy regularly to stay informed about our latest privacy protection measures.

5. Contact Us

If you have any questions or suggestions about this privacy policy, please contact us through:

  • GitHub Issues: https://github.com/xingBaGan/atujii/issues
