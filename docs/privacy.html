隐私政策

1. 信息收集

图迹是一款本地图片管理工具，我们承诺：

  • 所有图片和视频文件仅保存在您的本地设备上
  • 不会收集或上传您的个人数据到任何服务器
  • AI标签功能完全在本地运行，不会将图片发送到云端

2. 数据存储

您的数据安全对我们至关重要：

  • 所有媒体文件保持在您的本地存储中
  • 应用配置和设置仅保存在本地
  • 不会在云端存储任何用户数据

3. 权限使用

图迹需要以下权限来提供完整功能：

  • 文件系统访问权限：用于读取和管理您的媒体文件
  • 网络权限：仅用于软件更新检查
  • 系统通知权限：用于显示操作完成提醒

4. 更新说明

我们可能会定期更新本隐私政策。更新后的政策将在本页面发布，并标注最后更新时间。建议您定期查看本政策以了解最新的隐私保护措施。

5. 联系我们

如果您对本隐私政策有任何疑问或建议，请通过以下方式联系我们：

  • GitHub Issues: https://github.com/xingBaGan/atujii/issues
